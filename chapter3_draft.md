# 第3章 结果与讨论

## 3.1 冷空气爆发和非冷空气爆发条件下气溶胶粒径分布特征

基于第2.1.3节筛选的四类典型个例，本文分别从冬季冷空气爆发、夏季冷空气爆发、冬季非冷空气爆发和夏季非冷空气爆发四种情景出发，对2020年ACTIVATE飞行期间观测到的气溶胶粒径分布特征进行分析。综合图3.1至图3.4可以看出，不同季节和不同天气背景下粒径谱形态存在明显差异，其中冷空气爆发条件下更容易出现以细粒子为主的增强信号，而非冷空气爆发条件下粒径分布整体更平缓，更多表现为老化气溶胶主导的宽峰结构。这表明天气形势与季节背景共同调控了海洋边界层中颗粒物的生成、增长及其后续演化。

### 3.1.1 冬季冷空气爆发事件的粒径分布特征

图3.1给出了2020年2月28日和3月1日四个航次的冬季冷空气爆发事件粒径分布。总体来看，冬季冷空气爆发条件下粒径分布在小粒径段表现出显著增强，主要峰值集中在10-50 nm范围内，且不同航次之间的变率较大，反映出该类天气过程中粒子生成和早期增长具有较强的瞬变特征。尤其是在部分航次中，10 nm以下粒子浓度明显抬升，同时20-40 nm粒径段维持较高数浓度，说明观测区域内可能存在较新近发生的新粒子生成事件，或处于新生粒子向艾特肯模态持续增长的阶段。

与之相比，100 nm以上粒子虽然仍有一定贡献，但其谱峰强度相对有限，说明冷空气爆发期间数浓度的增加主要来自细粒子段，而非积聚模态粒子的被动累积。结合北大西洋西部区域冬季冷空气爆发的气象特征，可以推测该现象与以下过程密切相关：一是冷空气南下伴随的强风和海气湍流混合增强了海洋前体物的垂直输送；二是低温条件降低了低挥发性蒸气的饱和蒸气压，有利于成核及早期凝结增长；三是锋后降水和空气团置换降低了背景气溶胶表面积，从而减弱凝结汇，为新生粒子存活和增长提供了更有利的环境。该结果与前人关于冷空气爆发有利于海洋环境中新粒子生成发生的认识基本一致，也支持了Namdari等（2024）关于最冷月份新粒子生成最活跃的结论。

### 3.1.2 夏季冷空气爆发事件的粒径分布特征

图3.2展示了2020年9月21日至23日夏季冷空气爆发个例的粒径分布。与冬季冷空气爆发相比，夏季冷空气爆发期间粒径谱仍表现出冷空气过程对细粒子增长的促进作用，但总体上峰位有所后移，更多集中在30-100 nm范围内，粒径分布也更加平滑。这说明夏季冷空气爆发虽然同样可以为颗粒生成和增长提供有利条件，但其作用形式可能更偏向于促进已有新生粒子进一步增长，而不是像冬季那样频繁出现极小粒径段的显著突增。

从单个航次来看，夏季冷空气爆发事件中部分谱型呈现双峰或宽峰特征，表明观测区域内同时存在不同演化阶段的气溶胶群体：较小粒径段可能对应新近生成或尚未完全长大的颗粒，较大粒径段则可能代表经历一定凝结和凝并过程后的艾特肯模态或积聚模态颗粒。夏季较强的太阳辐射和活跃的光化学反应本应有利于前体物氧化，但海洋边界层在暖季通常具有更高的水汽含量和更复杂的云过程，可能增强颗粒的湿清除与云内处理，使得夏季冷空气爆发下的新粒子生成信号较冬季更分散，表现为中等粒径段的持续抬升而非极端尖锐的小粒径峰值。

### 3.1.3 冬季非冷空气爆发事件的粒径分布特征

图3.3所示的2020年3月11日至12日冬季非冷空气爆发事件中，粒径分布整体较为平缓，主峰主要位于30-120 nm范围内，10 nm以下粒子浓度明显低于冬季冷空气爆发个例，表明在缺少强冷空气输送和锋后清除作用时，大气中不具备同样有利的新粒子生成环境。此时观测到的气溶胶更可能以区域输送和老化积累后的粒子为主，其粒径谱呈现出较为连续的宽峰结构，而非突出的新生粒子峰。

值得注意的是，尽管冬季非冷空气爆发条件下仍然存在一定的细粒子贡献，但其峰值强度和谱宽均弱于冷空气爆发情景。这说明冬季本身的低温背景并不足以单独触发强烈的新粒子生成事件，关键仍在于冷空气爆发所带来的空气团转换、海气交换增强以及背景凝结汇降低等综合作用。换言之，季节背景决定了新粒子生成的潜在有利条件，而是否发生强信号事件，还取决于具体天气尺度过程的触发。

### 3.1.4 夏季非冷空气爆发事件的粒径分布特征

图3.4给出了2020年9月29日至30日夏季非冷空气爆发事件的粒径分布。该类个例整体表现为数浓度偏低、粒径峰位偏大，谱型以50-150 nm附近的宽峰为主，显示暖季稳定海洋边界层背景下颗粒物更倾向于处于相对成熟的增长阶段。从图中可以看出，超细粒子段并未出现持续且显著的增强，说明在无冷空气扰动的情况下，海洋环境中前体物供应、混合输送和凝结汇条件难以协同达到触发明显新粒子生成的阈值。

此外，个别航次在约100 nm附近出现较强峰值，可能与局地污染输送、云处理后残留粒子增加或仪器拼接区间附近的谱形突变有关，但从总体分布特征来看，夏季非冷空气爆发个例依然以相对老化的艾特肯模态和积聚模态粒子占主导。与冬季冷空气爆发情景相比，这类粒径分布更加支持“无明显新粒子生成、以背景气溶胶维持为主”的判断。

### 3.1.5 四类事件的对比分析

综合四类案例可见，冷空气爆发与否是影响粒径分布形态的关键因子，而季节变化进一步调节了这一影响的表现方式。冬季冷空气爆发个例中，小粒径段浓度提升最为显著，显示出最典型的新粒子生成或初期增长特征；夏季冷空气爆发虽同样存在细粒子增强，但峰值粒径相对偏大，反映出颗粒增长信号强于初始成核信号；两类非冷空气爆发个例则均以较宽的中等粒径峰为主，说明气溶胶总体更趋于老化。

这一结果说明，北大西洋西部区域海洋边界层中的新粒子生成并非持续均匀发生，而是受到天气过程强烈调制。冷空气爆发通过降低凝结汇、增强前体物供给和促进垂直混合，为新粒子生成创造“窗口期”；同时，冬季更低的温度又进一步放大了这一效应，因此冬季冷空气爆发成为最有利于新粒子生成发生的典型环境。这一认识与现有研究对该区域“最冷月份最强、云顶附近更活跃”的结论具有一致性，也为后续理解海洋云凝结核来源及其气候效应提供了观测依据。

## 3.2 四类案例的颗粒浓度时间变化特征

图3.5展示了不同飞行航次中每半小时颗粒最大浓度及其对应粒径的变化特征。总体来看，各航次之间颗粒最大浓度差异较大，说明ACTIVATE观测区域内颗粒物数浓度具有显著的时间变化和个例依赖性。在部分航次中，最大浓度远高于5×10^3 cm^-3，同时对应粒径多小于50 nm，表明这些高浓度事件主要由超细粒子或新生粒子贡献，而并非由大粒径背景气溶胶单纯累积所致。这一特征进一步支持了前述冷空气爆发条件下存在显著新粒子生成活动的判断。

从不同类型个例的对比来看，较高的颗粒浓度峰值主要出现在冬季相关航次，说明冷季背景下大气更容易形成高数浓度、低粒径的颗粒群体；而夏季和非冷空气爆发条件下，颗粒最大浓度整体较低，且对应粒径常向更大粒径端偏移，反映出颗粒物更可能已经历一定程度的增长和老化。也就是说，图3.5不仅体现了数浓度强度的变化，也从峰值粒径角度揭示了不同天气背景下颗粒物所处的演化阶段差异。

值得强调的是，最大浓度与对应粒径之间呈现出较明显的反向关系，即浓度越高时往往对应更小的峰值粒径；而当峰值粒径增大至50 nm以上时，最大浓度通常趋于降低。这说明在研究区域内，强颗粒事件更可能与新粒子生成及早期增长相关，而不是单纯由长距离输送的老化积聚模态粒子控制。对海洋环境而言，这种“小粒径、高数浓度”的事件具有更重要的气候意义，因为若其后续能够继续增长到云凝结核尺度，则可能显著改变海洋边界层云滴数浓度和云反照率。

## 3.3 结果讨论与可能机制

本研究结果表明，北大西洋西部区域新粒子生成活动具有显著的天气尺度依赖性，其中冷空气爆发是影响粒径谱形态和颗粒数浓度的重要外部强迫。冷空气爆发发生时，极地或高纬冷空气快速入侵海洋边界层，在增强海气交换和边界层湍流混合的同时，也通过降温与湿清除降低凝结汇，从而形成有利于新粒子生成和存活的环境。特别是在冬季，这一效应叠加低温背景后表现得更为突出，因此冬季冷空气爆发个例中出现最明显的小粒径高浓度特征。

另一方面，非冷空气爆发背景下虽然并非完全不存在细粒子，但由于缺乏显著的空气团置换和海洋前体物脉冲输送，粒径分布更容易表现为宽峰型老化结构。这说明海洋新粒子生成并不是一个仅由前体物浓度决定的过程，还受到边界层稳定度、背景气溶胶负荷、湿清除历史以及云过程等多因素共同调制。结合已有研究可知，海洋环境中的硫酸、氨、胺、碘氧化物及有机物等都可能参与成核过程，而冷空气爆发为这些前体物的释放、输送与转化提供了有利条件。

因此，从区域气候意义看，本研究支持这样一种判断：北大西洋西部海洋边界层中的新粒子生成具有明显的事件驱动特征，冬季冷空气爆发是其最重要的触发场景之一。这不仅有助于解释该区域云凝结核来源的时空差异，也说明在气溶胶-云相互作用研究中，仅用季节平均或长期平均条件描述海洋气溶胶过程可能低估极端天气事件的贡献。

## 3.4 本章小结

本章基于ACTIVATE项目四类典型案例，对冷空气爆发和非冷空气爆发条件下的气溶胶粒径分布及颗粒浓度变化特征进行了分析。结果表明：

1. 冬季冷空气爆发事件中，小粒径段颗粒浓度增强最显著，粒径峰值多集中在10-50 nm范围内，表现出典型的新粒子生成或初期增长特征。
2. 夏季冷空气爆发事件同样存在细粒子增强，但谱峰整体向更大粒径偏移，说明其更偏向于促进粒子增长而非触发极强的初始成核。
3. 冬季和夏季非冷空气爆发个例均以30-150 nm范围内的宽峰分布为主，显示背景气溶胶老化特征更为明显，新粒子生成信号相对较弱。
4. 颗粒最大浓度较高的航次通常对应较小的峰值粒径，说明高数浓度事件主要与超细粒子增强有关；冷空气爆发尤其是冬季冷空气爆发，是造成这一现象的关键天气背景。

总体而言，冷空气爆发显著改变了北大西洋西部区域海洋边界层中气溶胶的粒径结构和数浓度水平，并可能通过影响云凝结核补给进一步作用于海洋低云性质。这为后续开展成核机制诊断和模型参数化评估提供了观测基础。
